# Code Book for tidydata.txt

## Data

- Simple txt format
- 180 rows by 563 columns
- 2 ID fields and 561 features


## Variables

### ID Fields

-	Variable	1	subject
	  - The participant ID
	  - Value from 1 to 30

-	Variable	2	activity
	  - Value:
	    - LAYING
	    - SITTING
	    - STANDING
	    - WALKING
	    - WALKING_DOWNSTAIRS
	    - WALKING_UPSTAIRS
